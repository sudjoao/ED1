- Cada espaço de memória só armazena 1 byte.

## Ponteiros

- São variáveis que armazenam endereço de memória. Ex:

```exemplo de ponteiro
int i=5;
int * pti=&i;
```

### Declaração de ponteiros

- **tipo \* nome**;
